# NomNom Recipes
A webpage to search for your desired recipes via ingredient, dish name or cuisine type. 

## Technologies Used
1. JavaScript
2. CSS
3. HTML
4. JQuery
5. AJAX
6. Google Fonts
7. EDAMAM API
8. IMGUR images
